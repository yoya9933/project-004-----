# 2026-07-08 風險評估工具原型建置紀錄

## 目標

建立工程違約金風險評估工具原型，以本機靜態網頁展示 120 件既有案件的數值預測結果。工具聚焦期末展示，不支援新案件手動輸入。

- `04_執行稿/build_risk_assessment_tool_data.py`
- `06_交付物/risk_assessment_tool/index.html`

- `06_交付物/risk_assessment_tool/assets/app.js`

- `06_交付物/risk_assessment_tool/data/risk_tool_status.json`
- `README.md`

## 工具功能

- 搜尋案號、案名與法院。
- 依年度、時間切分與風險等級篩選案件。
- 顯示酌減機率、預測准許比例、預測酌減率與酌減區間。
- 顯示 Ridge、Lasso 與 mean baseline 的准許比例比較。
- 顯示模型重要特徵貢獻。
- 顯示每案前 5 件 RAG 相似案例。
- 保留 AI 假設版標註與人工查核提醒。

## 風險等級規則

- 高：酌減機率 >= 70%，或預測酌減率 >= 50%。
- 中：酌減機率 >= 45%，或預測酌減率 >= 25%。
- 低：未達高/中門檻。

## 啟動方式

```powershell
cd .\06_交付物\risk_assessment_tool
python -m http.server 8000
```

開啟 `http://localhost:8000`。

## 限制

- 本工具目前只展示既有 120 件案件，不做新案件手動輸入。
