# 期末展示示範流程

本流程使用既有靜態網頁原型展示一件 2025 測試集案件，重點是說明「AI 假設版標註、模型回測、RAG 相似案例」如何串成工程違約金風險評估工具。展示時避免宣稱工具可取代法律判斷。

## 1. 展示前準備

在專案根目錄執行：

```powershell
cd .\06_交付物\risk_assessment_tool
python -m http.server 8000
```

開啟：

```text
http://localhost:8000
```

若 8000 port 已被占用，可改用：

```powershell
python -m http.server 8018
```

並開啟：

```text
http://localhost:8018
```

## 2. 建議示範案件

建議搜尋這個案號：

```text
TPHV,113,建上,46,20250610,1
```

案件資訊：

| 項目 | 內容 |
|---|---|
| 年度 | 2025 |
| 切分 | 測試集 2025 |
| 法院 | 臺灣高等法院民事 |
| 案名 | 返還違約金 |
| 風險等級 | 高 |
| AI 假設標註狀態 | `manualChecked = ai_assumed` |

選這件的原因：

- 屬於 2025 測試集，符合回測展示邏輯。
- 分類、比例、模型比較、特徵貢獻、RAG 相似案例都有完整資料。
- 預測結果容易說明：模型預測有酌減風險，且預測酌減幅度偏高。

## 3. 現場展示步驟

### 步驟一：選案件

操作：

1. 在搜尋框輸入 `TPHV,113,建上,46,20250610,1`。
2. 點選該案件。

講法：

> 我們先選一件 2025 年測試集案件。模型訓練時只看 2021 到 2023 年資料，2025 年案件用來模擬「用過去案件輔助判斷未來案件」。

### 步驟二：看是否酌減預測

畫面重點：

| 指標 | 值 |
|---|---|
| 預測是否酌減 | 是 |
| 酌減機率 | 約 80.0% |
| 風險等級 | 高 |

講法：

> 第一個模型是分類模型，任務是預測法院是否可能酌減違約金。這件案子的酌減機率約 80%，因此系統標示為高風險。

補充：

> 這裡的高風險不是法律結論，而是模型根據目前 AI 假設版標註與特徵計算出的風險提示。

### 步驟三：看准許比例與酌減率

畫面重點：

| 指標 | 值 |
|---|---|
| Ridge 預測准許比例 | 約 46.7% |
| Ridge 預測酌減率 | 約 53.3% |
| 預測酌減區間 | 中度酌減 |
| AI 假設標註准許比例 | 約 14.9% |
| AI 假設標註區間 | 大幅酌減 |

講法：

> 第二個模型不直接預測原始判賠金額，而是預測法院准許比例，也就是 `allowed_penalty / claimed_penalty`。這樣可以避免大金額案件過度影響模型。

> 在這件案子中，Ridge 模型預測法院可能准許約 46.7%，換算成酌減率約 53.3%，屬於中度酌減。AI 假設標註中的實際比例約 14.9%，表示模型方向有抓到酌減，但幅度仍有誤差。

### 步驟四：看模型比較

畫面重點：

| 模型 | 預測准許比例 |
|---|---|
| Mean baseline | 約 68.2% |
| Ridge Regression | 約 46.7% |
| Lasso Regression | 約 55.0% |

講法：

> 這裡不是只看單一模型，而是和 baseline 比較。Mean baseline 代表如果完全不看案件特徵，只用訓練集平均值會怎麼預測；Ridge 和 Lasso 則會納入案件特徵。

> 這件案子中，Ridge 和 Lasso 都比平均 baseline 預測出更高的酌減幅度，代表模型有受到案件特徵影響。

### 步驟五：看重要特徵

畫面重點：

分類模型可講：

- 業主可歸責爭點
- 逾期天數
- 每日違約金
- AI 候選金額數
- 業主已使用成果爭點

比例模型可講：

- 業主可歸責爭點
- AI 候選金額數
- 逾期天數
- 主張違約金 log
- 展延申請爭點

講法：

> 這個區塊不是說法院一定採納這些理由，而是說模型在計算時，哪些特徵對預測值影響比較大。它的功能是讓使用者知道模型不是黑箱，至少可以追溯模型主要被哪些欄位推動。

提醒：

> 特徵貢獻是統計模型解釋，不等於法律上的因果關係。

### 步驟六：看檢索增強生成相似案例

畫面重點：

第一件相似案例：

| 項目 | 內容 |
|---|---|
| 案號 | `ILDV,112,建,5,20240605,1` |
| 年度 | 2024 |
| 法院 | 臺灣宜蘭地方法院民事 |
| 案名 | 返還違約金 |
| 相似度 | 約 0.9707 |
| 共同詞 | 過高、酌減、工期、逾期、逾期違約金、展延、公共工程、民法第252條 |

講法：

> RAG 的用途不是直接套用相似案例結論，而是幫使用者快速找到事實與爭點相近的案件。使用者可以從相似案例進一步回到原判決全文，比對金額、展延、可歸責與法院理由。

第二件相似案例也可以提：

| 項目 | 內容 |
|---|---|
| 案號 | `TPDV,114,建,6,20250912,1` |
| 年度 | 2025 |
| 案名 | 返還不當得利 |
| 相似度 | 約 0.9510 |

### 步驟七：說明人工智慧假設版限制

最後一定要講：

> 目前這個工具是 AI 假設版標註下的流程展示。也就是說，金額、是否酌減、爭點欄位還沒有逐案完成正式人工審核。因此它適合作為資料分析與風險提示原型，不是法律意見，也不能直接預測法院判決。

可補充：

> 如果要走向正式研究，下一步會優先人工查核 `claimed_penalty`、`allowed_penalty`、`is_reduced`、`remaining_ratio` 與法院核心理由，並重新回測模型。

## 4. 一分鐘濃縮版講稿

> 我們的工具先從工程違約金判決資料中建立 120 件案件標註表，再用 2021 到 2023 年案件訓練模型，2024 年驗證，2025 年測試，2026 年作為最新年度外部檢查。
>
> 這件 2025 年測試集案件中，分類模型預測法院酌減機率約 80%，因此標示為高風險。比例模型進一步預測法院可能准許約 46.7% 的違約金，換算酌減率約 53.3%。
>
> 工具不只顯示預測值，也顯示 baseline、Ridge、Lasso 的比較，並列出影響模型判斷的重要特徵。最後，RAG 會找出相似工程違約金案例，讓使用者回到原判決全文查核。
>
> 目前版本採 AI 假設版標註，因此定位是期末原型與流程驗證，不是正式法律意見。
