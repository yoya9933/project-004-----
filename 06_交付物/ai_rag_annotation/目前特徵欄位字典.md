# 目前特徵欄位字典

本文件說明目前 120 件主線模型使用的特徵欄位。舊的 26 筆 TPOT 測試欄位已移除，不再作為目前模型入口。

## 主資料表

- 來源表：`06_交付物/ai_rag_annotation/annotation_workbook.csv`
- 分類模型特徵矩陣：`06_交付物/is_reduced_classification/feature_matrix.csv`
- 比例模型資料表：`06_交付物/reduction_ratio_model/ratio_model_frame.csv`

## 人工正式爭點特徵

這 6 欄是目前主線的正式爭點特徵。值為 `0` 或 `1`；只有在法院理由或重要事實明確涉及時才應填 `1`。

| 欄位 | 模型欄位 | 說明 |
|---|---|---|
| `issue_owner_fault` | `x_issue_owner_fault` | 是否涉及業主可歸責，例如設計、圖說、場地、協力、指示或驗收流程造成工期影響。 |
| `issue_contractor_fault` | `x_issue_contractor_fault` | 是否涉及承包商可歸責，例如施工遲延、管理不當、未依約履行或瑕疵補正遲延。 |
| `issue_extension_request` | `x_issue_extension_request` | 是否涉及展延工期、展期申請、免計工期或工期認定爭議。 |
| `issue_actual_damage_unclear` | `x_issue_actual_damage_unclear` | 是否涉及實際損害不明、損害難證、損害偏低，或法院以損害程度作為酌減考量。 |
| `issue_partial_completion` | `x_issue_partial_completion` | 是否涉及工程已部分完成、部分可使用、部分驗收或遲延只影響部分工項。 |
| `issue_used_by_owner` | `x_issue_used_by_owner` | 是否涉及業主已使用、接收或受益於工程成果。 |

## AI 建議爭點特徵

這 6 欄是 AI/RAG 標註包先行提示的同組爭點。它們可輔助人工查核，也會在目前展示模型中保留為特徵；正式研究時應回到原判決全文核對。

| 欄位 | 模型欄位 | 對應正式欄位 |
|---|---|---|
| `ai_suggest_issue_owner_fault` | `x_ai_issue_owner_fault` | `issue_owner_fault` |
| `ai_suggest_issue_contractor_fault` | `x_ai_issue_contractor_fault` | `issue_contractor_fault` |
| `ai_suggest_issue_extension_request` | `x_ai_issue_extension_request` | `issue_extension_request` |
| `ai_suggest_issue_actual_damage_unclear` | `x_ai_issue_actual_damage_unclear` | `issue_actual_damage_unclear` |
| `ai_suggest_issue_partial_completion` | `x_ai_issue_partial_completion` | `issue_partial_completion` |
| `ai_suggest_issue_used_by_owner` | `x_ai_issue_used_by_owner` | `issue_used_by_owner` |

## 數值事實特徵

| 原始欄位 | 模型欄位 | 說明 |
|---|---|---|
| `contract_price` | `x_log_contract_price` | 契約總價，模型使用 log 轉換值。 |
| `claimed_penalty` | `x_log_claimed_penalty` | 業主主張或扣罰的違約金，模型使用 log 轉換值。 |
| `contract_price`、`claimed_penalty` | `x_claim_to_contract_ratio` | 主張違約金占契約總價比例。 |
| `delay_days` | `x_delay_days` | 法院認定或標註的逾期天數。 |
| `claimed_penalty`、`delay_days` | `x_penalty_per_delay_day` | 主張違約金除以逾期天數。 |

## 標籤與目標欄位

這些欄位用來建立目標變數或檢查模型結果，不應被當成預測特徵直接餵給模型。

| 欄位 | 說明 |
|---|---|
| `is_reduced` | 法院是否酌減違約金；分類模型目標。 |
| `allowed_penalty` | 法院最後准許違約金；只用於推導比例目標，不作為模型特徵。 |
| `remaining_ratio` | `allowed_penalty / claimed_penalty`；比例模型主要目標。 |
| `reduction_rate` | `1 - remaining_ratio`；酌減率。 |
| `reduction_bucket` | 由 `remaining_ratio` 分段後的酌減程度。 |
| `key_reason` | 法院酌減或不酌減的核心理由摘要；供人工閱讀與展示，不作為目前線性特徵。 |

## 關鍵詞與候選輔助欄位

下列欄位主要用於篩選、baseline 或品質檢查。因為部分欄位可能帶有判決結論語彙，正式預測未判決案件時應謹慎使用。

| 欄位 | 說明 |
|---|---|
| `x_money_candidate_count` | AI 候選金額數量。 |
| `x_delay_candidate_count` | AI 候選逾期天數數量。 |
| `x_relevance_score` | 關鍵詞篩選階段的相關度分數。 |
| `x_penalty_term_count` | 違約金相關詞命中數。 |
| `x_delay_term_count` | 工期/逾期相關詞命中數。 |
| `x_strong_reduction_term_count` | 酌減、過高、民法第 252 條等強酌減詞命中數。 |
| `x_has_mfa_252` | 是否命中民法第 252 條相關語彙。 |
| `x_has_discretion` | 是否命中酌減語彙。 |
| `x_has_over_high` | 是否命中過高語彙。 |
| `x_keyword_rule_predict_reduced` | 關鍵詞規則 baseline 的酌減預測。 |
