# 六項正式特徵相關性分析設計

## 目標

在根目錄 `streamlit_app.py` 新增「特徵相關性」頁籤，使用
`06_交付物/120_判決主要特徵值總表.csv` 的六項二元正式特徵，
同時呈現：

1. 六項特徵彼此的相關矩陣。
2. 六項特徵分別與「是否酌減」及「酌減率」的相關性。

`正式特徵值摘要` 僅供文字說明，不納入數值計算。

## 資料來源與欄位

資料來源固定為：

`06_交付物/120_判決主要特徵值總表.csv`

分析欄位：

- `正式特徵_業主可歸責`
- `正式特徵_承包商可歸責`
- `正式特徵_展延免計工期爭議`
- `正式特徵_實際損害不明偏低`
- `正式特徵_部分完成部分驗收`
- `正式特徵_業主已使用受益`

目標欄位：

- `是否酌減`：`是` 轉換為 `1`，`否` 轉換為 `0`。
- `酌減率`：定義為
  `1 - 剩餘比例 = (主張違約金 - 法院准許違約金) / 主張違約金`。
  CSV 內以 `0` 到 `1` 的小數保存；例如主張 100 萬元、法院准許
  40 萬元時，剩餘比例為 `0.4`，酌減率為 `0.6`。只有主張違約金
  大於 `0`，且法院准許違約金介於 `0` 與主張違約金之間時才視為有效；
  空白、無法解析或不符合範圍的資料不參與該項計算。

## 計算方式

六項正式特徵先以數值方式解析，只接受 `0`、`1` 或可等價轉換的布林值。

- 特徵與特徵：使用 Pearson 相關係數。兩個變數均為二元時，此值等同
  Phi coefficient。
- 特徵與「是否酌減」：使用 Pearson 相關係數，等同兩個二元變數的
  Phi coefficient。
- 特徵與「酌減率」：使用 Pearson 相關係數，等同二元特徵與連續變數的
  point-biserial correlation。

本功能只呈現描述性相關係數，不計算顯著性檢定或因果結論。

若任一欄位只有一個唯一值，其標準差為零，相關係數應保留為缺值並顯示
「無變異，無法計算」，不得以 `0` 取代。現有資料中的
`正式特徵_部分完成部分驗收` 120 筆皆為 `1`，屬於此情況。

## 程式結構

在 `streamlit_app.py` 內新增彼此獨立的資料處理與呈現函式：

1. 載入並驗證正式特徵 CSV。
2. 將六項特徵與兩項目標轉成分析用 DataFrame。
3. 計算特徵相關矩陣。
4. 計算特徵與兩項目標的相關係數、有效樣本數及 0/1 件數。
5. 呈現相關性頁籤。

計算函式不依賴 Streamlit 狀態，使驗證腳本可直接測試。

## 介面

現有三個頁籤後新增第四個頁籤「特徵相關性」，內容依序為：

1. 資料說明與有效樣本數。
2. 六項特徵的 0/1 件數表。
3. 六項特徵彼此的相關矩陣，以 Pandas Styler 色階表呈現。
4. 特徵與「是否酌減」的相關係數表及水平長條圖。
5. 特徵與「酌減率」的相關係數表及水平長條圖。
6. 對相關係數方向、強度及無變異欄位的限制說明。

介面沿用現有 Streamlit 與 Pandas，不新增 Plotly、Altair 或 SciPy 依賴。

## 錯誤處理

- CSV 不存在：只在相關性頁籤顯示錯誤，不中止其他頁籤。
- 缺少必要欄位：列出缺少的欄名，不執行相關計算。
- 欄位含無效值：轉為缺值並從對應的成對計算排除。
- 有效樣本不足或欄位無變異：顯示「無法計算」及原因。
