# 移除「限制說明」分頁設計

## 目標

從 Streamlit 主畫面移除「限制說明」分頁及該分頁內的人工查核內容。

## 修改範圍

- 將主畫面的三個分頁改為「案件儀表板」與「資料總覽」兩個分頁。
- 刪除 `tab_limits` 對應的畫面渲染區塊。
- 保留 `render_notice()`，因為沒有符合篩選條件的案件時仍會使用該提示。
- 不修改資料載入、模型訓練、案件儀表板、資料總覽或側邊欄行為。

## 驗證

- 執行 Python 語法編譯檢查。
- 搜尋確認主畫面不再建立「限制說明」分頁。
- 確認 `render_notice()` 仍由無選取案件的分支呼叫。
